Ce repertoire est destine a recueillir des exemples de specifications 
formelles B a utiliser comme jeux de tests pour les outils.

Afin de preserver une organisation minimale, il est convenu de respecter 
dans la mesure du possible les conventions suivantes:

Ce repertoire est consacre au langage B «prime'.

Le langage B prime est definit dans le rapport technique suivant:
 Georges Mariano, Jean-Louis Boulanger et Bruno Tatibouet
 Revisiting B language syntax
 Laboratoire CEDRIC-CNAM
 1999
 TR 99-07

Chaque contribution doit presenter une structure uniforme de projet :

	* un repertoire maitre donnant le nom de la contribution

	* un repertoire 'spec' contenant les specifications formelles

	* les autres repertoires nécessaires
