Ce répertoire est destiné à recueillir des exemples de spécifications 
formelles B à utiliser comme jeux de tests pour les outils.


Ce répertoire est consacré aux tests des structures du langage B.


Chaque contribution doit présenter une structure uniforme de projet :

	* un répertoire maitre donnant le nom de la contribution

	* un répertoire 'spec' contenant les spécifications formelles

	* les autres répertoires nécessaires

	* un certain ensemble de noms de répertoires auxiliaires 
	devraient être considérés comme ´réservés"
